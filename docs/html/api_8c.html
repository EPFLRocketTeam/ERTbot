Parses a JSON string to extract the value associated with a given key.

Parameters
[in]jsonPointer to the JSON string to be parsed.
[in]keyPointer to the key whose associated value is to be retrieved.
Returns
char* Pointer to the extracted string value. The caller is responsible for freeing the allocated memory. If the key is not found or the JSON is malformed, the function returns NULL.

This function searches for the specified key within the JSON string, locates the associated value, and returns a dynamically allocated copy of that value. Special handling is implemented for the key "content" to correctly parse the JSON structure.

size_t writeCallback ( void *  data,
size_t  size,

void *  clientp 
)

Function called by the API to handle data received in response to an HTTP request.

This callback function is used to store the data received from an HTTP request in a memory buffer. It is typically called multiple times as data is received in chunks.

Parameters
[in]dataPointer to the delivered data.
[in]sizeSize in bytes of each element of data.
[in]nmembNumber of elements in the data.
[in]clientpPointer to a user-defined data structure (struct memory) that holds the response buffer.
Returns
size_t The number of bytes successfully written to the memory buffer. If memory allocation fails, the function returns 0.

This function dynamically resizes the memory buffer to accommodate the incoming data. The received data is appended to the buffer, and the buffer is null-terminated to ensure it remains a valid C-string.
