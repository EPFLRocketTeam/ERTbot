Extracts a substring from the input string based on specified start and end delimiters.

Parameters
[in]inputStringPointer to the input string from which the substring will be extracted.
[in]startDelimiterPointer to the start delimiter string.
[in]endDelimiterPointer to the end delimiter string.
[in]includeStartBoolean indicating whether to include the start delimiter in the extracted substring.
[in]includeEndBoolean indicating whether to include the end delimiter in the extracted substring.
Returns
char* Pointer to a newly allocated string containing the extracted substring. Returns NULL if any of the delimiters are not found, or if memory allocation fails. The caller is responsible for freeing the allocated memory.

This function searches for the startDelimiter and endDelimiter within the inputString. It extracts the text between these delimiters based on the includeStart and includeEnd flags. If includeEnd is true, the text up to and including the endDelimiter is extracted; otherwise, the text up to but not including the endDelimiter is extracted.

Note
The function allocates memory for the resulting substring. Ensure to free the allocated memory to avoid memory leaks.
Warning
Ensure that inputString, startDelimiter, and endDelimiter are not NULL. Handle potential memory allocation failure and check for null pointers in practice.
